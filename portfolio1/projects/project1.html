Smart Translator

Project Overview

The Smart Translator is a tool designed to translate YouTube videos into Tamil. It extracts audio from the video, transcribes the speech, translates the text into Tamil, and synthesizes the translated text back into speech. This project was developed to help non-English speakers access content in their native language.

Features

  • Audio extraction from YouTube videos.
  • Speech-to-text transcription using advanced APIs.
  • Text translation into Tamil using machine learning models.
  • Text-to-speech synthesis for the translated text.
  • User-friendly interface for seamless interaction.

Technologies Used

API Integration PostgreSQL Django Whisper API Embart50

Challenges & Solutions

One of the main challenges was handling large audio files efficiently. To address this, I implemented chunk-based processing, where the audio is split into smaller segments for transcription and translation. Additionally, optimizing the translation speed was achieved by using pre-trained models and caching frequently used translations.
